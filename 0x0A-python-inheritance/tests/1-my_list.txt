Using ``print_sorted`` method
-------------------

First import the ``1-my_list`` module:

>>> MyList = __import__('1-my_list').MyList

>>> my_list = MyList()

>>> my_list.append(9)
>>> my_list.append(6)
>>> my_list.append(3)
>>> print(my_list)

[1, 3, 6, 9]
